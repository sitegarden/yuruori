ひとこと

タイプは決めつけるものというより、 キャラや自分を眺めるためのメモとして使うと楽しいです。

このページについて

TYPE
16タイプっぽいものを、ゆるく楽しむ場所

このページは、性格タイプやキャラ傾向をメモするためのページです。 創作キャラの整理、診断サイト作り、タイプ別の曲や雰囲気作りにも使えます。

あくまで個人的なメモなので、人の性格を決めつけるためのページではありません。

管理人メモ

好きな見方

タイプを「正解探し」として見るより、 その人のノリ、考え方、得意な動き方を見るための材料として使うのが好きです。

創作との相性

キャラの話し方、行動パターン、恋愛のすれ違い、チーム内の役割を考える時にかなり便利です。

診断サイトとの相性

結果ページごとに文章や称号を変えられるので、診断サイトの題材としてもかなり面白いです。

タイプ一覧メモ

ENTP系

発想、ツッコミ、実験、議論、変化。 ネタにすると強いタイプ。

ひらめき / 口が回る / 遊び心
ESTP系

行動、勢い、体感、勝負、ノリ。 ゲームや動きのある診断に合うタイプ。

即行動 / 勝負感 / 現場力
ENFP系

明るさ、広がり、感情の熱量、人とのつながり。 物語を動かしやすいタイプ。

自由 / 熱量 / 可能性
INFP系

世界観、こだわり、繊細さ、自分だけの価値観。 創作キャラにかなり使いやすいタイプ。

理想 / 物語 / 内面
INTP系

分析、構造、理屈、観察。 システムや世界設定を考える時に強いタイプ。

分析 / 探究 / 論理
ENFJ系

導く力、空気作り、人を巻き込む力。 主人公の味方やリーダー役に合うタイプ。

支援 / 影響力 / まとめ役
ISFP系

感性、雰囲気、好き嫌い、自然体。 絵や音楽、空気感と相性がいいタイプ。

感性 / 自然体 / 好き
ENTJ系

目標、戦略、指揮、前進。 強いリーダーやラスボス寄りにも使いやすいタイプ。

戦略 / 指揮 / 達成

創作に使うなら

キャラ作り 口調、行動、怒り方、照れ方、恋愛の動き方を考えるヒントにできます。
曲作り タイプごとに似合うテンポ、声、歌詞、ジャンルを考えるとかなり楽しいです。
診断作り 質問、結果文、称号、シェア文まで作れるのでサイト向きです。
ゲーム作り タイプ別のカード、称号、相性、パーティ編成などに応用できます。
